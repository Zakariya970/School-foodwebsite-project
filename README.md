# College Food Ordering Website

This project is a website designed to allow students to order food online and collect it from a college shop.

## Features
- Online food ordering system
- Simple and clear user interface
- Menu display for students
- Designed to improve efficiency and reduce queues

## Technologies Used
- HTML
- CSS

## Purpose
This project was created as part of a college assignment to simulate a real-world digital solution for food ordering.
